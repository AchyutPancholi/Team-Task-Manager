# Team Task Manager - Implementation Plan

## Tech Stack
- Backend: Node.js + Express.js
- Database: PostgreSQL (Railway native)
- Auth: JWT + bcrypt
- Frontend: Vanilla HTML/CSS/JS (premium design)

## Structure
- /backend (Express API)
- /frontend (Static HTML/CSS/JS)
- Dockerfile for Railway

## Features
1. Auth: Signup/Login with JWT
2. Role-based: Admin / Member
3. Projects: CRUD, team assignment
4. Tasks: Create, assign, status tracking
5. Dashboard: Stats, overdue tasks
